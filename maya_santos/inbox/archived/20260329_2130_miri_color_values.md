## Expression Sheet Notes

1. **Eye highlight position: upper-LEFT.** This is a shared visual DNA marker with Luma (same family). It must stay upper-left on every expression.
2. **Cheek blush (#D4956B):** This is her permanent resting-state blush — present on ALL expressions including neutral. It is NOT an expression-reactive blush. For expressions with added emotional blush (embarrassment, warmth), layer additional blush on top.
3. **Dual-Blush Rule:** When Miri appears in the same frame as Luma's excited blush, her own blush fades to ~7.5% feel (30% of normal). This is a character tell. Your expression sheet should note this as a frame-context modifier, not a separate expression pose.
4. **Eye iris (#8B5E3C) is the same amber family as Luma's iris (#C87D3E) but darker and richer.** At small sizes, ensure there is sufficient contrast between iris and pupil (#1A0F0A).
5. **Line color: #3B2820 (Deep Cocoa)** — show-wide standard. Previous drafts referenced #4A2810 — that value is superseded and must NOT be used.
