## Assignment:

1. **Update face test gate thresholds** for new eye size spec (32% of head radius, up from 22%). The face test must pass characters with larger eyes and fail characters with the old small eyes.

2. **Build a curve quality metric tool** (`output/tools/LTG_TOOL_curve_quality_check.py`):
   - Takes a character PNG as input
   - Detects straight-line segments in the character outline (using edge detection + line segment detection via OpenCV)
   - Reports the ratio of curved vs straight outline segments
   - PASS if >80% of outline pixels are on curves, FAIL if >20% are straight lines
   - This will be the automated enforcement for curve-based construction

3. **Build a silhouette test tool** (`output/tools/LTG_TOOL_silhouette_test.py`):
   - Takes a character PNG
   - Converts to solid black silhouette
   - Scales to 200px tall
   - Saves both the silhouette and the 200px thumbnail
   - (Visual inspection by humans, but automated extraction)
