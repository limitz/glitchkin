## Task 2 — Motion Spec Lint Tool (Section 8, precritique QA)

**New tool:** `output/tools/LTG_TOOL_motion_spec_lint.py`

Pixel-analysis QA tool — no Claude vision API, no cost.

**Checks:**
1. Image dimensions ≤ 1280px (FAIL if violated)
2. Image not blank/solid (FAIL if violated)
3. Panel count matches expected (WARN if mismatch)
4. Annotation zone pixel occupancy per panel (WARN if < 4%)
5. Beat badge detection per panel (WARN if absent)
6. BEAT_COLOR cyan family in annotation zones ≥ 50% of panels (WARN)

**Integration:** Section 8 added to `LTG_TOOL_precritique_qa.py` → upgraded to v2.5.0. Tools chain is now [1/8]–[8/8]. `_make_snapshot()`, `build_report()`, and `main()` all updated.

- `LTG_CHAR_byte_motion.png` (4 panels): panel count PASS, size PASS — heuristic WARNs same reason
