## P2 — CRT Glow Profile Extraction Tool

Built `LTG_TOOL_glow_profile_extract.py` (v1.0.0). Extracts from CRT reference photos:
1. Gaussian fit of phosphor glow radius (FWHM)
2. Color temperature of glow spill (McCamy's formula)
3. Falloff curve parameters (sigma, amplitude, baseline, R-squared)

Ran on 17 CRT reference photos. 10 good fits (R^2 > 0.5). Results:
- **Recommended sigma_frac = 0.1165** (Gaussian sigma as fraction of canvas diagonal)
- **Recommended FWHM = 27.4% of diagonal** (how far the glow extends)
- **Peak amplitude = 49.4** (intensity above ambient, 0--255 scale)
- **Baseline = 4.1** (ambient spill floor)
- **Color temperature = 13070K median** (heavily cool/blue — expected for phosphor emission)

These parameters can feed directly into kitchen/living room CRT glow calibration (Hana Okonkwo's generators).
