## Delivered

### `LTG_TOOL_draw_shoulder_arm.py` (new shared module)
- Location: `output/tools/LTG_TOOL_draw_shoulder_arm.py`
- Reusable shoulder-to-arm drawing with proper shoulder involvement (Takeshi C15-C46 persistent critique)
- Implements the Shoulder Involvement Rule (image-rules.md C47) and shoulder_mechanics_reference_c47.md

### Public API
1. **`draw_shoulder_arm()`** — Main entry point. Draws complete shoulder + arm assembly:
   - Computes shifted shoulder point based on arm angle (rise, spread, cross-body)
   - Draws deltoid bump with per-clothing style (hoodie=fabric bunch, cardigan=crease, fitted=arc)
   - Draws two-segment arm (upper + forearm with elbow bend) or single-segment
   - Draws hand
   - Returns (hand_x, hand_y)

2. **`compute_shoulder_shift()`** — Pure computation of shifted shoulder point. Can be used independently for torso polygon construction without drawing.

3. **`shoulder_polyline()`** — Returns torso top-edge polyline with both shoulder points shifted. Drop-in replacement for straight top edges.

4. **`ShoulderArmStyle`** — Dataclass for per-character style params (arm_fill, clothing type, deltoid_radius, line_width, two_segment flag, elbow_bend_factor).

- `hoodie` (Luma): wider ellipse deltoid = fabric bunch at sleeve top

- `fitted` (Cosmo): clean compressed arc = visible muscle silhouette
