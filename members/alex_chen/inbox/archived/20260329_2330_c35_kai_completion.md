## Task 2 ‚Äî LTG_TOOL_spec_extractor_v001.py

Built and registered. Key capabilities:
- Parses character `.md` spec files: Luma, Cosmo, Miri, Byte, Glitch
- Extracts: head_ratio, eye_coeff, hex colors with labels, degree constants, line weights, element counts
- Output: structured Python dict (JSON-serialisable)
- API: `extract_spec(char_name)`, `extract_all()`, `format_spec_report()`
- CLI: `python LTG_TOOL_spec_extractor_v001.py [char|all] [--json] [--save-dir]`

This tool is designed to make it easier to keep `char_spec_lint_v001.py` in sync with the spec docs. Future: `LTG_TOOL_spec_sync_check_v001.py` (ideabox idea this cycle) would compare extractor output vs. lint constants automatically.
