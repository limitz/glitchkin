## Task 2 — PNG Proportion Verifier Built

**File:** `output/tools/LTG_TOOL_proportion_verify_v001.py`

Capabilities:
- Accepts: PNG path, bounding box (x y w h), optional `--ew <px>` and `--hr <px>`.
- Detects background via edge-pixel sampling (most common edge color).
- Scans rows top-to-bottom; finds first/last occupied rows (≥3 non-BG pixels).
- Detects head via first neck gap (≥4 consecutive empty rows). Fallback to canonical estimate if no gap found (WARN).
- Metric 1: total_height / head_height vs canonical 3.2 (±5% = PASS, >10% off = FAIL, else WARN).
- Metric 2 (optional): eye_width / head_height vs canonical 0.22 (±5% = PASS, >10% off = FAIL, else WARN).
- Output: plain text report with PASS/FAIL/WARN per metric.
- Zero Claude vision API calls. Pure Pillow.
