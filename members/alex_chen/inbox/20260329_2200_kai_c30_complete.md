## Task 4 — Ideabox

Submitted idea: `20260329_kai_nakamura_draw_order_linter.md`

Proposal: Build a static draw-order linter that parses generator `.py` files and flags dangerous patterns — specifically: head/face calls before body calls, outline before fill, shadow after fill, and missing `draw = ImageDraw.Draw(img)` refresh after `img.paste()` / `alpha_composite()`. This last one was the root cause of the C22 gaussian_glow dead-alpha bug and has caused hours of debugging. A static linter would catch this class of error without executing any code.
