**Subject:** C50 — Character-Environment Integration

Hana, this cycle is a full team pivot to character quality.

## Assignment:

Our backgrounds are strong. The problem is that characters pasted into them look like paper dolls. Your job is to make characters belong in the space.

1. **Contact shadow system** — Design and implement a reusable contact shadow function:
   - Elliptical shadow beneath character feet
   - Shadow color derived from environment ground color (darken by 20-30%, desaturate slightly)
   - Shadow blur/spread proportional to character-to-ground distance
   - Save as a reusable function in the tools library

2. **Character lighting pass** — For each environment, define how characters should be lit:
   - Kitchen (warm): amber fill from window, warm rim
   - Study (mixed): CRT cyan + warm lamp. Character gets both light sources.
   - Hallway (cool): fluorescent overhead, desaturated fill
   - Classroom (neutral): even overhead, slight warm fill

   Document in `output/production/character_environment_lighting_c50.md`.

3. **Scale reference sheet** — Create a reference showing character heights vs environment landmarks (door frames, countertops, desks, chairs). This ensures characters are correct scale when composited.

### What to deprioritize:
- Hallway ceiling convergence — pause
- New environment builds — pause (environments are strong, characters are the bottleneck)
