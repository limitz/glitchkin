# Ideabox

Each cycle, every team member contributes **at least one idea**. Ideas can be anything:
- Something you could do that would help another team member
- A pain point you have with someone else's tools
- A general improvement to the project, pipeline, or workflow

## How to submit
Create a file named `YYYYMMDD_[your_name]_[short_title].md` in this directory.

## Format
```
**Author:** [name]
**Cycle:** [number]
**Date:** [YYYY-MM-DD]
**Idea:** [one paragraph max]
**Benefits:** [who would this help and how]
```

## Review
At the end of each cycle, the **producer** reviews all ideas and decides which ones to follow up on.
Ideas that get actioned are moved to `ideabox/actioned/`.
