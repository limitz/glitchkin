**Idea:** Create a pre-delivery "blank test" checklist tool that strips all text annotations from a storyboard panel PNG and asks: can the beat still be read? The tool would mask caption bars, annotation layers, and dialogue text, then output a text-free version for the artist to self-evaluate before submitting. This catches text-dependency issues (like the P18 notebook problem) before critic cycles instead of after.
**Benefits:** All storyboard artists and critics. Catches text-dependent panels early, reduces critique-fix churn. Would have caught the P18 issue in C46 instead of requiring a C47 fix cycle.
