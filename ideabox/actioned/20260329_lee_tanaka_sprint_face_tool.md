**Idea:** Build a small Python script — `LTG_TOOL_character_face_test_v001.py` — that renders a character head at sprint/wide-shot scale (head_r = 20–25px) and draws candidate facial expressions at that scale. It would output a contact sheet of ~6–8 face variants (different eye sizes, brow angles, mouth shapes) as a 600×400 reference. This would let the team QA that a face expression is legible before embedding it in a full style frame generator. Currently we can only verify face readability by running the full SF generator, inspecting the output, and editing. A standalone face-at-scale test tool would catch "invisible at sprint scale" problems before they go into the pipeline.
**Benefits:** Saves Rin and Maya multiple full-generator iteration cycles when working on SF02 character faces. Reduces cost of expression micro-tuning. The tool generates reference-only output — no PNG goes into production, just into the review loop. Anyone on the team can run it to check a proposed expression tweak before committing it to a generator.
