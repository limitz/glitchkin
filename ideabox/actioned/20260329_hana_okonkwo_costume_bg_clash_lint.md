**Idea:** Build a "character-background color clash detector" tool that, given an environment generator's color constants and a character color model, automatically computes the Euclidean RGB distance and value separation between every costume color and every background fill color. Flag any pair below a minimum separation threshold (e.g., RGB distance < 50 or value delta < 20). This would have caught the Cosmo cardigan/locker merge issue in v002 before it shipped — the tool would have flagged LOCKER_LAV (168,155,191) as a clash with Cosmo RW-08 Dusty Lavender (168,155,191) immediately. Input: two Python files (env generator + character color model doc); output: a report table of all close-call pairs with severity (CLASH / WARN / OK).
**Benefits:** Prevents figure-ground failures before assets are generated. Saves iteration cycles. Particularly useful for ensemble scenes where multiple characters appear in one environment — each new character added to the cast should be auto-checked against all existing environments. Saves Maya and the storyboard team from discovering background merge issues during compositing.
