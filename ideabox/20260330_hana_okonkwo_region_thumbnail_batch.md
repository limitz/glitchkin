**Idea:** Extend the thumbnail preview tool to accept multiple named regions in a JSON file, so artists can annotate an entire set of story-critical details (figurines, prop placements, signage) across an environment in one pass and get a contact sheet of all of them side by side at thumbnail scale. This would be useful when handing off a new environment to Lee Tanaka for storyboarding — he can see immediately which details survive the thumbnail and which need size/contrast reinforcement before panels are drawn.
**Benefits:** Saves Lee Tanaka and any storyboard artist from discovering late that a prop they staged around is invisible in pitch materials. Also helps me validate new environments more systematically than spot-checking by eye.
