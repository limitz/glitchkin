## TASK 1 — Build `output/tools/ltg_render_lib.py` [PRIORITY 1]

This is the project's shared rendering utility library. All generators can import it. Build the following functions (each with a proper docstring):

```python
def perlin_noise_texture(width, height, scale=50, seed=42, octaves=4, alpha=80):
    """Generate an RGBA noise overlay for texture effects.
    Returns a PIL Image (RGBA) of size (width, height).
    Uses layered sine/cosine approximation or simple value noise if numpy available."""

def gaussian_glow(draw_target, center, radius, color, max_alpha=120, steps=8):
    """Paint a radial glow effect by drawing concentric ellipses with decreasing alpha.
    draw_target: ImageDraw object
    center: (cx, cy) tuple
    radius: outer glow radius
    color: RGB tuple
    """

def light_shaft(img, apex, base_left, base_right, color, max_alpha=60, feather_passes=2):
    """Draw a light shaft as a filled polygon with feathered (blurred) edges.
    img: PIL Image (RGBA or RGB)
    apex: (x, y) top point of shaft
    base_left, base_right: bottom corners
    Returns modified image.
    """

def dust_motes(draw, bounds, count=20, seed=42, color=(255,252,240), alpha_range=(50,90)):
    """Scatter small dust particles within bounds rectangle.
    bounds: (x0, y0, x1, y1)
    Each mote is a small circle (radius 1-4px) with alpha in alpha_range.
    """

def catenary_wire(draw, p0, p1, sag_ratio=0.05, color=(40,35,30), widths=(3,1)):
    """Draw a wire with natural catenary sag between two points.
    widths: (main_width, secondary_width) for layered wire effect.
    """

def scanline_overlay(img, spacing=4, alpha=15, seed=42):
    """Add subtle scanline texture (horizontal lines) for CRT/retro effect.
    Returns modified image.
    """

def vignette(img, strength=80):
    """Add a subtle dark vignette around the image edges.
    Returns modified image.
    """
```

Save as: `output/tools/ltg_render_lib.py`
