# STATEMENT OF WORK — CYCLE 01
## "Luma & the Glitchkin"

**Date:** 2026-03-29
**Cycle:** 01 (Initial Setup & Foundation)
**Prepared by:** Alex Chen, Art Director

---

## Summary

Cycle 01 was the project's foundation cycle. The primary objective was to establish the visual and production language of the show before any character, background, or storyboard work could begin. No animation-ready assets were produced this cycle; instead, the deliverables are the documents and structure that make all future asset creation possible and consistent.

---

## Deliverables Completed This Cycle

### 1. Style Guide
- **File:** `/home/wipkat/team/output/style_guide.md`
- **Created by:** Alex Chen (Art Director)
- **Version:** 1.0
- **Status:** Complete — ready for critic review

**Contents:**
- Show concept overview
- Art style direction and style family (visual references established)
- Character design principles: shape language table, proportions, appeal checklist, face/expression standards, costume notes
- Color palette specifications: Real World palette (6 swatches with hex codes), Glitch palette (6 swatches with hex codes), mood palette guides
- Background art direction: style, level-of-detail table by shot type, perspective approach, key environment descriptions
- Line work standards: character lines, background lines, glitch lines
- Visual dos and don'ts
- Output directory structure

### 2. Production Bible
- **File:** `/home/wipkat/team/output/production/production_bible.md`
- **Created by:** Alex Chen (Art Director)
- **Version:** 1.0
- **Status:** Complete — ready for critic review

**Contents:**
- Show logline and elevator pitch
- Target audience (age range 7–12, platform considerations)
- Tone and content guidelines (tonal pillars, what's in, what's out)
- Episode format (22-minute runtime, 3-act structure with segment breakdown)
- Main cast overview: Luma, Cosmo, Byte (personality, visual hook, shape language, relationships, arc)
- World overview: Millbrook/Real World and Glitch Layer (rules of each)
- Recurring elements: Grandma Miri, the Glitchkin as a species, pixel confetti, Millbrook as a character
- Visual style summary (cross-referenced with style guide)
- Production standards: file formats, resolution, naming conventions, directory structure, workflow

## Notes for Cycle 02

1. **Character designs are the Cycle 02 critical path.** All other Cycle 02 work either requires or benefits from having the main cast locked.
2. The CRT monitor prop (`prop_crt_monitor_miri_v01`) is a hero prop that will appear in nearly every episode. It should be designed to the highest fidelity — it is the show's narrative anchor.
3. Sam Kowalski should reference the hex codes from the Style Guide exactly — no approximations. Color consistency is non-negotiable.
4. Lee Tanaka should begin thumbnail planning independently while awaiting final character sheets. Thumbnails do not require locked designs; they establish composition and pacing.
5. Critics' feedback from this cycle should be incorporated into updated versions of the Style Guide and/or Production Bible before Cycle 02 asset work is finalized.
