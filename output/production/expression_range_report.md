## Metric Definitions
- **FRPD** (Face Region Pixel Delta): Mean absolute pixel difference in face. 0=identical, 1=totally different.
- **SCI** (Structural Change Index): Fraction of face pixels with per-channel diff > noise threshold.
- **ERS** (Expression Range Score): Average FRPD across all pairs. Aggregate expressiveness.
- **FRPD thresholds**: FAIL < 0.03, WARN < 0.08, PASS >= 0.08
- **SCI thresholds**: FAIL < 0.05, WARN < 0.15, PASS >= 0.15
- **ERS thresholds**: FAIL < 0.05, WARN < 0.1, PASS >= 0.1
