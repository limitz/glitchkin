**Note 1 — Wall Color Divergence (Minor):**
The Cycle 14 color key specified Warm Cream `#FAF0DC` as the primary large-area wall fill.
The generator uses WALL_SAGE `#8A9E8A` — a muted green-grey institutional sage.
This divergence originates from `millbrook_school.md` which specifies sage green classroom walls
as part of Millbrook's school building design language. The generator correctly references
`millbrook_school.md` as its color authority.

**Ruling:** Generator is CORRECT per `millbrook_school.md`. The Cycle 14 color key was written
before the classroom background spec was finalized and defaulted to Warm Cream. The color key
should be understood as an approximation, not a binding wall-color spec.
The sage green wall is a valid institutional warm-neutral that keeps the space warm-reading without
Glitch contamination. Sage Green `#7A9E7E` (RW-06) is in the Real World palette — close enough.
The generator's `#8A9E8A` is a slightly more muted/greyer version of RW-06 (7 points lower on R;
same G; same B). Within production tolerance.

**Action:** None required. Jordan's `millbrook_school.md` reference overrides the color key's
approximate Warm Cream guess for walls. Carry forward: update classroom color key note for future
cycles to specify Sage Green `#8A9E8A` as the canonical classroom wall color.
