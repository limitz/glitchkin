## Why these files are here

These storyboard panel images were generated in early production cycles before
the LTG naming convention was established. Each has a corresponding
`LTG_SB_*` equivalent in `output/storyboards/panels/`.

**These files are superseded. Use the LTG-named panel images for all production work.**
