## Dual-Blush Rule — Critical Production Note

When Miri appears in the same frame as Luma in an excited state:

- **Normal state:** Permanent cheek blush at #D4956B, 25% opacity feel.
- **"Pride Override" state (Luma excited blush active in frame):** Permanent cheek blush fades to approximately 7.5% opacity feel (30% of the normal 25%) — barely visible warmth. Miri is suppressing the blush to look composed.
- **Activation:** Same frame as Luma's excited blush appears. No delay.
- **Deactivation:** 8 frames after Luma's excited blush disappears. Miri's blush returns to 25%.
- **This is a character tell:** The suppression signals Miri is privately proud but not showing it yet.

## Glasses Frame — Production Clarification (Cycle 17)

The original color model did not specify a glasses frame color. Based on visual DNA and production intent:

- **Glasses frame:** `#8A7A70` (Warm Gray) — identical to eyebrow color.
- Rationale: thin wire-frame glasses should visually recede. Using the same Warm Gray as the eyebrows keeps the glasses from competing with the face. Glasses and eyebrows share a value family; at production scale this reads as "understated and quiet."
- Shadow companion: none specified — the glasses frame is a thin graphic stroke shape, not a fill zone.

## Lighting Rules — Grandma Miri (Cycle 17 Addition)

Miri exists exclusively in warm/domestic Real World environments. The following rules govern her rendering at all times:

1. **NO Glitch palette colors on Miri — ever.** Electric Cyan (#00F0FF), Byte Teal (#00D4E8), Hot Magenta (#FF1AB4), UV Purple (#7B2FBE), Acid Green (#7FFF00), Void Black (#0A0A14) must not appear on Miri's body, clothing, or cast shadows at any time, in any scene.
2. **Key light is always warm.** Miri's key is Soft Gold (#E8C95A, RW-02) or Sunlit Amber (#D4923A, RW-03). In den scenes: Ochre Brick lamp glow (#B8944A, RW-13) is acceptable as a secondary warm source. In nighttime scenes with Miri, use a reduced warm key — not a cool key.
3. **Ambient fill is Warm Cream (#FAF0DC, RW-01) or Dusty Lavender (#A89BBF, RW-08).** Dusty Lavender is acceptable as a secondary ambient/sky bounce only when a window is visible. It must be used at low influence — Miri's silhouette should always read warm.
4. **Skin shadow: always #6A3A1E (Dark Sienna) under warm key.** Use Cool Skin Shadow (#7A5A7A, RW-10c) only if a nighttime scene has NO warm light source at all. Den scenes always have the lamp.
5. **Monitor glow in den scenes: use RW-12 (Muted Teal #5B8C8A) as the screen ambient — NOT Electric Cyan.** The CRT is an analog domestic object, not a Glitch device. Its glow is aged and desaturated.
6. **Cheek blush (#D4956B) is always present at 25% opacity feel.** It is a permanent resting-state feature, not an expression beat. See Dual-Blush Rule above for the Pride Override exception.
