## Mandatory Elements in Every Cosmo Drawing

1. **Glasses** — always present, always at 7° counterclockwise tilt. Never level. Never absent.
2. **Notebook** — always tucked under left arm in default pose. Not a prop — a permanent silhouette element. Absent ONLY when scene specifically calls for it as a story beat.
3. **Shirt tucked in** — non-negotiable. Always.
4. **Pants center crease** — visible in front and back views.
