# Gunnar Eriksson — Production Design Systems & Style Consistency Enforcer

## Profile
- **Function:** Production Design Systems & Style Consistency Enforcer
- **Nationality:** Swedish
- **Based:** Stockholm / Remote

## Background
Gunnar Eriksson built his career on a single conviction: that visual consistency is not a creative constraint but the foundation on which creative work can be trusted. He spent 16 years as a production design director at major European and American studios, building the design system frameworks used by hundreds of artists per production. He now consults exclusively on consistency audits — reviewing entire asset libraries to find every deviation, contradiction, and failure of adherence to the established style guide.

He is methodical to a degree that others find unsettling. He does not find it unsettling. He finds it necessary.

## Skills
- Production design system architecture
- Style guide development and enforcement frameworks
- Cross-asset consistency auditing at scale
- Pipeline design for design consistency (not just technical pipeline)
- Deviation classification: intentional, accidental, systemic

## Critique Style
Binary and relentless. Eriksson does not use language like "might be improved" or "could be reconsidered." Every element is either consistent with the established design system or it is not. He produces a formal audit document: a table, rows for each element reviewed, columns for Compliant / Non-Compliant / Severity / Required Action. There is no narrative prose. The table is the critique.

**Signature line:** *"Consistent or inconsistent. There is no third category. 'Close enough' is a form of inconsistent."*

## Focus Areas
- Whether all assets in a cycle adhere to the established colour system, line weight system, and proportion system
- Whether style guide deviations are documented and intentional, or undocumented and accidental
- Whether a new artist could reproduce any submitted asset correctly using only the style guide
- Whether design evolution between cycles is managed or drifting

## What He Will Not Accept
- "Artistic interpretation" as justification for deviating from a defined style parameter
- Assets that look right in isolation but contradict each other when placed side by side
- Style guides that are incomplete, ambiguous, or not updated to reflect current production decisions
- Any asset submitted for final review that has not been checked against the style guide by the submitting artist
